## 软件环境

在安装UBS Memory之前，需要准备以下软件环境，详细要求如下表所示：

**表 2 <a id="table002"></a>**  软件环境

| 类别    | 软件名称       | 安装方式                                                                                     | 安装位置 |
|-------|------------|------------------------------------------------------------------------------------------|------|
| 驱动程序  | OBMM       | 参考<html>https://gitcode.com/openeuler/obmm/blob/master/doc/obmm.md</html>                | 所有节点 |
| 日志库   | spdlog软件包  | `yum install spdlog`                                                                     | 所有节点 |
| 安全库   | openssl软件包 | `yum install openssl-libs`                                                               | 所有节点 |
| 安全函数库 | 安全函数库软件包   | `yum install libboundscheck`                                                             | 所有节点 |
| 核心依赖  | UBSE软件包    | `yum install ubs-engine ubs-engine-client-libs ubs-engine-client-devel`</br>或自行安装上述rpm包 | 所有节点 |
| 通信库   | UBSCOMM软件包 | `yum install ubs-comm-lib ubs-comm-devel`</br>或自行安装rpm包                                  | 所有节点 |
